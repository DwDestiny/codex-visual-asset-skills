# 5 套风格候选

## 业务冷静型

- 候选图：`style-board-business-calm.svg`
- 生图 prompt：`Create a mini style board for a calm business presentation. Light warm gray background, restrained green accent, subtle orange highlights, clean process cards, editable chart feeling, no logos, no readable fake text, 16:9.`
- 验收：应能一眼看出封面节奏、内容页结构、图表语言和透明素材使用方向。
